============
Installation
============


Stable release
--------------

To install foo.bir, run this command in your terminal:

.. code-block:: console

    $ pip install foo.bir

This is the preferred method to install foo.bir, as it
will always install the most recent stable release.

To create the documentation, you can install foo.bir
with the extra ``docs``:

.. code-block:: console

    $ pip install foo.bir[docs]


If you don't have `pip <https://pip.pypa.io>`_ installed, this
`Python installation guide <https://docs.python-guide.org/starting/installation/>`_
can guide you through the process.

From sources
------------

The sources for foo.bir can be downloaded from the
`Github repo <https://github.com/veit/foo.bir>`_.

You can either clone the public repository:

.. code-block:: console

    $ git clone git://github.com/veit/foo.bir

Or download the `tarball <https://github.com/veit/foo.bir/tarball/main>`_:

.. code-block:: console

    $ curl  -OL https://github.com/veit/foo.bir/tarball/main

Once you have a copy of the source, you can install it with:

.. code-block:: console

    $ python -m pip install foo.bir
